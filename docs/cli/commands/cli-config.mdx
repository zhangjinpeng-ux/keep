设置 Keep 配置。

## Usage

```
Usage: keep config [OPTIONS] COMMAND [ARGS]...
```

## Options
* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  显示此帮助信息并退出。



## CLI Help

```
Usage: keep config [OPTIONS] COMMAND [ARGS]...

  Manage the config.

Options:
  --help  Show this message and exit.

Commands:
  new   create new config.
  show  show the current config.
```

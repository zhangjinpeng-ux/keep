创建新配置。

## Usage

```
Usage: keep config new [OPTIONS]...
```

## Options
* `interactive`:
  * Type: BOOL
  * Default: `True`
  * Usage: `--interactive`

  交互式创建配置。

* `url`:
  * Type: STRING
  * Default: `http://localhost:8080`
  * Usage: `--url`

  Keep 后端服务器的 URL。

* `api-key`:
  * Type: STRING
  * Default: ``
  * Usage: `--api-key`

  用于通过 Keep 认证的 API key。

* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  显示此帮助信息并退出。



## CLI Help

```
Usage: keep config new [OPTIONS]

  create new config.

Options:
  -u, --url TEXT      The url of the keep api
  -a, --api-key TEXT  The api key for keep
  -i, --interactive   Interactive mode creating keep config (default True)
  --help              Show this message and exit.
```

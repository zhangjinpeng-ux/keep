显示 Keep 配置。

## Usage

```
Usage: keep config show [OPTIONS]...
```

## Options
* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  显示此帮助信息并退出。



## CLI Help

```
Usage: keep config show [OPTIONS]

  show the current config.

Options:
  --help  Show this message and exit.
```

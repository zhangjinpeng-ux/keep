应用工作流。

## Usage

```
Usage: keep workflow apply [OPTIONS]
```

## Options
* `file` (REQUIRED):
  * Type: Path
  * Default: `none`
  * Usage: `--file
-f`

  工作流文件


* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  显示此帮助信息并退出。



## CLI Help

```
Usage: keep workflow apply [OPTIONS]

  Apply a workflow.

Options:
  -f, --file PATH  The workflow file  [required]
  --help           Show this message and exit.
```

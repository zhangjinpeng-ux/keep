获取库版本信息。

## Usage

```
Usage: keep version [OPTIONS]
```

## Options
* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  显示此帮助信息并退出。



## CLI Help

```
Usage: keep version [OPTIONS]

  Get the library version.

Options:
  --help  Show this message and exit.
```

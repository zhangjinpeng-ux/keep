启动 API。

## Usage

```
Usage: keep api [OPTIONS]
```

## Options
* `multi_tenant`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--multi-tenant`

  启用多租户模式


* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  显示此帮助信息并退出。



## CLI Help

```
Usage: keep api [OPTIONS]

  Start the API.

Options:
  --multi-tenant  Enable multi-tenant mode
  --help          Show this message and exit.
```

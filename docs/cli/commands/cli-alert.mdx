
# cli alert

管理告警。

## Usage

```
Usage: cli alert [OPTIONS] COMMAND [ARGS]...
```

## Options
* `help`:
  * Type: BOOL
  * Default: `false`
  * Usage: `--help`

  显示此帮助信息并退出。

Keep CLI 允许你通过命令行管理 Keep。

从[安装](/cli/installation) Keep CLI 和[运行工作流](/cli/commands/cli-run)开始。

### 环境变量

| 环境变量 | 用途 | 是否必需 | 默认值 | 有效选项 |
|:-------------------:|:-------:|:----------:|:-------------:|:-------------:|
| **KEEP_CLI_IGNORE_SSL** | 连接 KEEP API 时忽略 SSL | 否 | false | "true" 或 "false" |

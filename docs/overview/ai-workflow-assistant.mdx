AI 驱动的工作流构建器（请勿与[工作流中的 AI](./ai-in-workflows)混淆）是一个类似聊天的界面，用于使用自然语言构建工作流。
它以"人在回路"的范式工作，提出更改建议并仅在用户明确同意后才应用。
它简化了工作流构建流程，并帮助组织内更广泛的工程师群体采用工作流。

前往 "Workflows" -> "+ Create Workflow" 找到 AI 助手：

## 常见问题：

**使用的模型：** OpenAI、Keep 托管的模型或其他模型。<br/>
**数据流向：** 数据在 LLM 提供商和 Keep 之间共享，LLM 提供商可能因合同而异。

**告警管理**使团队能够有效地管理、监控和处理关键告警。

Keep 提供强大且用户友好的界面，让用户能够深入了解告警、在海量数据中过滤告警，并采取快速行动以维护系统健康。

<Frame width="100" height="200">
  <img height="10" src="/images/alert_table_1.png" />
</Frame>

与告警管理相关的所有内容都可以自定义：

1. **告警表格** - 查看和管理告警。
2. **搜索栏** - 使用 CEL 过滤告警，可保存为"自定义预设"。
3. **维度筛选** - 对告警进行切片和切块分析。
4. **列和时间** - 为您的预设自定义列和主题。

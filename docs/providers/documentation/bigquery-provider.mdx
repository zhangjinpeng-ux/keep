## 连接提供者

1. 创建一个 Google Cloud 项目并启用 BigQuery API。
2. 在 Google Cloud 项目中创建服务账户并下载 JSON 密钥文件。
3. 与服务账户共享必要的数据集。
4. 使用 `service_account_key`、`project_id` 和 `dataset` 配置你的提供者。

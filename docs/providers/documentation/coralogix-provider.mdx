## 概述

Coralogix 是一个现代可观察性平台，通过端到端监控，为你所有的日志、指标、追踪和安全事件提供全面的可见性。

## 连接 Coralogix 到 Keep

要将 Coralogix 连接到 Keep，你需要将其配置为来自 Coralogix 的 webhook。按照以下步骤设置集成：

1. 从 Coralogix 工具栏，导航到 Data Flow > Outbound Webhooks。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_1.png" />
</Frame>

2. 在 Outbound Webhooks 部分，点击 Generic Webhook。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_2.png" />
</Frame>

3. 点击 Add New。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_3.png" />
</Frame>

4. 输入 webhook 名称，将 URL 设置为 `https://api.keephq.dev/alerts/event/coralogix`。
5. 选择 HTTP 方法（POST）。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_4.png" />
</Frame>

6. 从 [Keep 设置](https://platform.keephq.dev/settings?selectedTab=api-key)生成一个具有 webhook 角色的 API 密钥。复制 API 密钥，在下一步中粘贴到请求标头中。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_5.png" />
</Frame>

7. 在 Coralogix webhook 配置中添加一个键为 "x-api-key"、值为 API 密钥的请求标头。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_6.png" />
</Frame>

8. 编辑 webhook 触发时发送的消息正文（可选）。
9. 保存配置。

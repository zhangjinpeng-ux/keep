## 连接提供者

### Kibana 主机

只需从浏览器 URL 栏中复制主机名：

<img src="/images/kibana/kibana_host.png" alt="Kibana Host" />

### API Key

要获取 Kibana API Key，请按照以下步骤操作：

1. 登录你的 Kibana 账户。
2. 点击 Stack Management
3. 点击 Security
4. 点击 API Keys

<img src="/images/kibana/api-keys.png" alt="Kibana API Keys" />

1. 点击右上角的 `Create API key` 按钮
2. 为 API 密钥起一个有指示性的名称（例如 keep-api-key）
3. 确保 `Restrict Permissions` 开关未打开
4. 在右下角，点击 `Create API key`

<img src="/images/kibana/create-api-key.png" alt="Create Kibana API Key" />

6. 复制新创建的编码 API 密钥，就设置好了！

<img src="/images/kibana/copy-created-key.png" alt="Copy Kibana API Key" />

## 指纹

Kibana 中的指纹就是告警实例 ID。

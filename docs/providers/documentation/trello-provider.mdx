## 连接 Provider

1. 前往 https://trello.com/power-ups/admin 创建自定义 power-up。
2. 创建新的 power-up，添加基本信息，如名称、电子邮件地址等。
3. 创建后，导航到 power-up 内部，前往 API Key 部分。
4. 点击 `Generate a new API key`，它将生成 API Key，用作 `api_key`。
5. 要生成 `api_token`，有手动生成 Token 的选项，点击该选项并授权应用程序。

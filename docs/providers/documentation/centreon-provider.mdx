## 连接提供者

1. Centreon 可以是 SaaS 或本地部署。你需要有一个正在运行的 Centreon 实例。
2. 转到 Administration > API Tokens 并为管理员用户创建新的令牌。
3. 使用你的 Centreon 实例的 URL 和 API 令牌来配置提供者。

## 有用链接

- [Centreon](https://www.centreon.com/)

## 注意

- Centreon 仅支持以下[主机状态](https://docs.centreon.com/docs/api/rest-api-v1/#realtime-information)（UP = 0, DOWN = 2, UNREA = 3）

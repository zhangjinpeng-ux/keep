## 连接 Provider

获取 Ntfy 访问令牌（仅适用于 Ntfy.sh）

1. 在 [Ntfy.sh](https://ntfy.sh/) 上创建账户。
2. 登录后，前往 [访问令牌](https://ntfy.sh/account) 页面。
3. 点击 `CREATE ACCESS TOKEN`。为其添加标签，选择令牌过期时间，然后点击 `CREATE TOKEN` 按钮。
4. 复制生成的令牌。这将作为 Provider 设置中的 `Ntfy Access Token` 使用。

自托管 Ntfy

1. 要自托管 Ntfy，您可以按照[此处的说明](https://docs.ntfy.sh/install/)进行操作。
2. 对于自托管 Ntfy，您需要在 Provider 设置中提供 `Ntfy Host URL`、`Ntfy Username` 和 `Ntfy Password`，而不是 `Ntfy Access Token`。
3. 为自托管 Ntfy 实例创建一个新用户，并在 Provider 设置中使用生成的用户名和密码。

订阅主题（适用于 Ntfy.sh 和自托管 Ntfy）

1. 登录您的 Ntfy.sh 账户。
2. 点击 `Subscribe to a topic` 按钮，为主题生成名称并订阅。
3. 复制生成的主题名称。这将作为 Provider 设置中的 `Ntfy Subcription Topic` 使用。
4. 保留主题并配置访问权限（需要 ntfy Pro）

## 连接提供者

在 GitLab 中创建你的 personal access token：
- 在左侧边栏中，选择你的头像。
- 选择 **Edit profile**。
- 在左侧边栏中，选择 **Access Tokens**。
- 选择 Add **new token**。
- 输入令牌的 **名称** 和 **过期日期**。
- 选择所需的范围。
- 选择 Create **personal access token**。

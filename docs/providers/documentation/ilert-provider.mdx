## 概述

ilert 提供者支持与 ilert API 的交互，允许管理事件和事件通知。这包括创建、更新和解决事件的能力，以及发送自定义事件通知。此提供者将 Keep 系统与 ilert 强大的告警和事件管理平台集成在一起。

## 连接提供者

要将 Keep 与 ilert 集成，请按照以下步骤操作：

1. 登录你的 ilert 账户。
2. 导航到账户设置下的 "Alert Sources"。
3. 专门为 Keep 创建一个新的告警源。
4. 记下为此告警源提供的 `ALERT-SOURCE-API-KEY`。

用于 Keep 集成的请求端点为：
(https://api.ilert.com/api/v1/events/keep/{ALERT-SOURCE-API-KEY})

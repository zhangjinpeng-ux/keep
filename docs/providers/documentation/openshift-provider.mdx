## 连接 Provider

要连接到 Openshift，请按照以下步骤操作：

1. 登录您的 Openshift 集群，创建一个具有所需角色的新服务账户。
2. 获取服务账户的令牌。
3. 使用令牌进行 Openshift 认证。

## 注意事项

- 此 Provider 允许您与 Openshift 交互以执行滚动重启操作。

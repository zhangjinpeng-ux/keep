## 连接 Provider

要使用 Opsgenie Provider，您需要提供 API 集成中的 API Key 和 Integration Name。您可以在 Settings -> Integrations -> Add integration 下创建 API 集成，搜索 API Integration。选择 API，为集成提供名称，然后点击继续。

您可以在 Settings -> Integrations -> Add integration 下创建集成密钥。

<Note>
  如果您使用的是免费版，集成密钥可以在 Teams ->
  Your team -> Integrations -> Add Integration (API) 下创建。
</Note>

访问 [Opsgenie API Integration](https://app.opsgenie.com/settings/integrations/create/api) 快速创建 API 集成。

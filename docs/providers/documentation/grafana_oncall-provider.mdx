## 连接提供者

要连接到 Grafana OnCall，你需要创建一个 API Token：

1. 登录你的 Grafana 账户。
2. 前往 "Alerts & IRM" -> OnCall。
3. 转到 **Settings** 页面。
4. 点击 **Create** 按钮并为你的令牌命名。
5. 复制令牌值并妥善保管。
6. 将令牌值添加到 Grafana Oncall Provider 配置的 `authentication` 部分。

## 注意事项

- 此提供者允许你与 Grafana OnCall 交互以创建告警。
- Keep 将在 Grafana OnCall 中创建名为 "Keep Integration" 的 "Webhook" 类型集成。

负载示例：

```json
{
    "alert_uid": "08d6891a-835c-e661-39fa-96b6a9e26552",
    "title": "The whole system is down",
    "image_url": "https://upload.wikimedia.org/wikipedia/commons/e/ee/Grumpy_Cat_by_Gage_Skidmore.jpg",
    "state": "alerting",
    "link_to_upstream_details": "https://en.wikipedia.org/wiki/Downtime",
    "message": "Smth happened. Oh no!"
}
```

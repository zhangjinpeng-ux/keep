## 概述

Monday Provider 实现了与 Monday.com 的无缝集成，Monday.com 是一个工作操作系统，帮助团队自信地运行项目和工作流。通过 Monday Provider，你可以向你的看板添加新的 pulse。

<AutoGeneratedSnippet />

#### Admin 标签页

如果你是 monday.com 账户的管理员用户，请按照以下步骤访问你的 API 令牌：

1. 登录你的 monday.com 账户。
2. 点击右上角的头像/个人资料图片。
3. 选择 Administration > Connections > API。
4. 复制你的个人令牌。请注意，你可以随时重新生成新令牌，但这样做会导致之前的任何令牌过期。

#### Developer 标签页

如果你是 monday.com 账户的成员用户或管理员，请按照以下步骤访问你的 API 令牌：

1. 登录你的 monday.com 账户。
2. 点击右上角的个人资料图片。
3. 选择 Developers。这将在另一个标签页中打开 Developer Center。
4. 点击 My Access Tokens > Show。
5. 复制你的个人令牌。请注意，你可以随时重新生成新令牌，但这样做会导致之前的任何令牌过期。

## 连接 Monday 到 Keep

1. 从 Monday 获取 API Token。
2. 在 Keep 中将 Monday 添加为提供者。
3. 为提供者命名，将 API Token 粘贴到 `Personal API Token` 字段中，然后点击 `Connect`。

## 如何使用？

1. 要向你的看板添加新的 pulse，你需要以下信息：
   - Board ID: 你要添加 pulse 的看板的 ID。
   - Group ID: 你要添加 pulse 的组的 ID。
   - Item Name: 你要添加的 pulse 的名称。
   - Column Values: 你要为 pulse 设置的列的值。
2. 在 monday.com 应用中打开你要添加 pulse 的看板。
3. 将鼠标悬停在侧面板中的看板名称上，点击出现的三个点，然后点击 ID 以复制看板 ID。
4. 将鼠标悬停在看板中的组名称上，点击出现的三个点，然后点击 Group ID 以复制组 ID。
5. Item Name 是你要添加的 pulse 的名称。
6. Column ID 和 Column Value 是你要为 pulse 设置的列的值。将鼠标悬停在看板中的列名称上，点击出现的三个点，然后点击 Column ID 以复制列 ID。列值是你要为该列设置的值。

## 连接提供者

1. 按照 [Google Cloud 服务账户指南](https://cloud.google.com/iam/docs/creating-managing-service-account-keys) 中的步骤获取 Google Cloud 凭据。
2. 确保你的服务账户具有管理 GKE 集群的必要权限（`roles/container.admin`）。
3. 在提供者配置中提供 `gcp_credentials`、`project_id` 和 `zone`。

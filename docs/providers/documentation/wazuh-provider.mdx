## 概述

Wazuh provider 实现了 Keep 与 Wazuh 之间的无缝集成。
它允许您通过自定义集成将 Wazuh 的告警发送到 Keep，使您能够更轻松地在一个地方跟踪安全相关的活动。

如果您想了解更多关于 Wazuh 自定义集成的信息，请参阅 [Wazuh 文档](https://documentation.wazuh.com/current/user-manual/manager/integration-with-external-apis.html#custom-integration)。

## 将 Wazuh 连接到 Keep

要将 Wazuh 连接到 Keep，您需要在 Wazuh 中将其配置为自定义集成。按照以下步骤设置集成：

1. Keep webhook 脚本需要安装在 Wazuh 服务器上。

2. 您可以使用以下命令下载 Keep webhook 脚本：

```bash
wget -O custom-keep.py https://github.com/keephq/keep/blob/main/keep/providers/wazuh_provider/custom-keep.py?raw=true
wget -O custom-keep https://github.com/keephq/keep/blob/main/keep/providers/wazuh_provider/custom-keep?raw=true
```

3. 将下载的脚本复制到 Wazuh 服务器上的以下路径：`/var/ossec/integrations/`，并设置正确的权限
```bash
cp custom-keep.py /var/ossec/integrations/custom-keep.py
cp custom-keep /var/ossec/integrations/custom-keep
chown root:wazuh custom-keep*
chmod 750 /var/ossec/integrations/custom-keep*
```

4. 获取 Keep 的 Webhook URL，即 `https://api.keephq.dev/alerts/event/wazuh`。

5. 获取 Keep 的 API Key，您可以在 [Keep 设置](https://platform.keephq.dev/settings?selectedTab=users&userSubTab=api-keys)中生成。

6. 在配置文件 `/var/ossec/etc/ossec.conf` 中设置新的集成块
```xml
<integration>
    <name>custom-keep</name>
    <level>10</level>
    <hook_url>PLACE_YOUR_KEEP_WEBHOOK_URL_HERE</hook_url>
    <api_key>PLACE_HERE_YOUR_API_KEY</api_key>
    <alert_format>json</alert_format>
</integration>
```
请参阅 [Wazuh 文档](https://documentation.wazuh.com/current/user-manual/manager/integration-with-external-apis.html#custom-integration)了解更多信息，并设置您感兴趣的 `level`。
7. 重启 `wazuh-manager`
```bash
$ systemctl restart wazuh-manager
```

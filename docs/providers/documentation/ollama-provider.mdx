<Tip>
  Ollama Provider 支持查询本地 Ollama 模型进行基于提示的交互。请确保您已在本地安装并运行 Ollama 以及所需的模型。
</Tip>

### **云环境限制**
此 Provider 在云环境中被禁用，只能在本地或自托管环境中使用。

## 连接 Provider

使用 Ollama Provider：

1. 从 [Ollama 官网](https://ollama.ai) 在您的系统上安装 Ollama。
2. 启动 Ollama 服务。
3. 使用 `ollama pull model-name` 拉取您所需的模型。
4. 在 Keep 配置中配置主机 URL。

## 前提条件

- Ollama 必须已在您的系统上安装并运行。
- 所需的模型必须已拉取并在您的 Ollama 安装中可用。
- Ollama API 必须可以从 Keep 运行的主机访问。

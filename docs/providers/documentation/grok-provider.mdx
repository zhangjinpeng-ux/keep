## 连接提供者

要连接到 Grok，你需要获取一个 API Key：

1. 在 X.AI 平台订阅 Grok。
2. 导航到你的 X.AI 账户设置中的 API 部分。
3. 为 Keep 生成新的 API Key。

在 Grok Provider 配置的 `authentication` 部分中使用生成的 API Key。

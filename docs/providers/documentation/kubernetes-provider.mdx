## 连接提供者

要连接到 Kubernetes，请按照以下步骤操作：

1. 在 Kubernetes 上创建一个 Service Account。
2. 创建 role/clusterrole 并使用 rolebinding/clusterrolebinding 绑定到 Service Account。
3. 获取 Service Account 的令牌。

## 注意事项

- 此提供者允许你与 Kubernetes 交互以执行滚动重启或列出 Pod 的操作。

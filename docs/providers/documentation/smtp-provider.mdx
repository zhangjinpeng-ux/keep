## 概述

SMTP Provider 允许您从 Keep 发送电子邮件。大多数电子邮件服务（如 Gmail、Yahoo、Mailgun 等）都提供 SMTP 服务器来发送电子邮件。您可以使用这些 SMTP 服务器从 Keep 发送电子邮件。

SMTP provider 支持纯文本和 HTML 格式的电子邮件，允许您创建丰富、样式化的电子邮件通知。

## 连接 SMTP Provider

1. 从您的电子邮件服务提供商获取 SMTP 凭据。例如：Gmail、Yahoo、Mailgun 等。
2. 在 Keep 中使用获取的凭据添加 SMTP Provider。
3. 将 SMTP Provider 与 Keep 连接。

## 电子邮件格式支持

SMTP provider 支持两种电子邮件格式：

### 纯文本电子邮件
使用 `body` 参数发送纯文本电子邮件：
```yaml
with:
  from_email: "sender@example.com"
  from_name: "Keep Alerts"
  to_email: "recipient@example.com"
  subject: "Alert Notification"
  body: "This is a plain text email notification."
```

### HTML 电子邮件
使用 `html` 参数发送 HTML 格式的电子邮件：
```yaml
with:
  from_email: "sender@example.com"
  from_name: "Keep Alerts"
  to_email: "recipient@example.com"
  subject: "Alert Notification"
  html: "<h1>Alert</h1><p>This is an <strong>HTML</strong> email notification.</p>"
```

当同时提供 `body` 和 `html` 时，HTML 内容优先。

## 多个收件人

您可以通过提供电子邮件地址列表向多个收件人发送电子邮件：
```yaml
with:
  to_email:
    - "recipient1@example.com"
    - "recipient2@example.com"
    - "recipient3@example.com"
```

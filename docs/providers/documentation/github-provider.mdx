## 连接提供者

1. 转到你的 GitHub 账户，导航到 **Settings > Developer Settings > Personal Access Tokens**。
2. 生成具有所需权限的令牌（例如，`repo`、`workflow` 等）。
3. 复制令牌并将其作为 `github_token` 在提供者配置中提供。

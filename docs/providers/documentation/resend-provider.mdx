# Resend Provider

ResendProvider 是一个实现 Resend API 的类，允许通过 Keep 发送电子邮件。

<AutoGeneratedSnippet />

## 连接 Provider
要通过 Keep 连接 Resend provider 并发送电子邮件，请按照以下步骤操作：

1. 获取 Resend API 密钥：访问 [Resend API Keys](https://resend.com/api-keys) 获取 API 密钥（如果您还没有）。
2. 使用获取的 API 密钥在系统中配置 Resend provider。
3. 使用以下 YAML 示例通过 Resend provider 发送电子邮件通知：

```yaml title=examples/alert_example.yml
# Send an email notification using the Resend provider.
alert:
  id: email-notification
  description: Send an email notification using Resend
  actions:
    - name: send-email
      provider:
        type: resend
        config: "{{ providers.resend-provider }}"
        with:
          _from: "sender@example.com"
          to: "recipient@example.com"
          subject: "Hello from Resend Provider"
          html: "<p>This is the email body.</p>"
```

### 启用 Incoming Webhook 应用程序

向频道发送 `/inc connect <connection name>` 以启用带有授权码的 webhook。应用程序将以 webhook url 和授权码作为响应。

<Tip>
您应该使用 "Full Format" Incoming Webhook Url，即以 `?format=full` 结尾的 URL。
</Tip>

## （可选）启用用户 JID 查找

消息可以选择性地包含 Zoom 用户 JID，用于在消息中标记特定的 Zoom 用户。
这非常有用，例如，当团队订阅了聊天频道，但成员只希望在被显式标记时收到通知。

### 创建 Zoom 应用程序

用户查找需要授权。创建一个仅限内部使用的 Zoom Server to Server OAuth 应用程序。

## 连接提供者
1. 确保你拥有具有必要[权限](https://docs.appdynamics.com/accounts/en/cisco-appdynamics-on-premises-user-management/roles-and-permissions)的 AppDynamics 账户。所需的基本权限是 `Account Owner` 或 `Administrator`。或者你可以创建一个账户[说明](https://docs.appdynamics.com/accounts/en/global-account-administration/access-management/manage-user-accounts)

## 提供者配置

1. [在此处](https://accounts.appdynamics.com/overview)查找你的账户名称。
2. 获取你希望在其中安装 webhook 的 AppDynamics 实例的 appId。
3. [在此处](https://accounts.appdynamics.com/overview)确定主机地址。

### Basic Auth 认证

1. 获取 AppDynamics 的 **Username** 和 **Password**
2. 转到 **Authentication** 部分下的 **Basic Auth** 标签页
3. 输入 **Username** 和 **Password**

## 连接提供者

1. 确保 **Install webhook** 已选中
2. 点击 **Connect**

## Webhook 集成修改

Webhook 集成将 Keep 作为告警监视器添加到 AppDynamics 实例中。可以在 "Alerts & Respond" 部分下找到它。
该集成自动获得 AppDynamics 中的以下权限范围：
- `administrator`
- `authenticated`

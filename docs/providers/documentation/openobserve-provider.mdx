## 连接 Provider

获取 OpenObserve 用户名和密码：
1. 了解如何安装和设置凭据：[此处](https://openobserve.ai/docs/quickstart/#self-hosted-installation)
2. 获取您希望安装 webhook 的 OpenObserve 实例的组织 ID。

## Webhook 集成修改

webhook 集成将 Keep 添加为 OpenObserve 实例中的告警监控器。可以在 "Alerts & Respond" 部分下找到。
该集成自动获得 OpenObserve 中以下权限范围的访问权限：
- `authenticated`

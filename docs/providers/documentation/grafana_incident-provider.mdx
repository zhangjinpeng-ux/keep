## 入门

1. 在你的 Grafana Cloud 实例中，点击左侧菜单中的 Alerts & IRM。
2. 点击 Incident 磁贴为你的 Grafana Cloud 实例启用该应用。
3. 一旦 Grafana Incident 启用，你组织中的用户即可访问它。

## 连接提供者

1. 启用 Grafana Incident 应用后，导航到 Adminstration > Users and access > Service Accounts。
2. 点击 Add Service Account 按钮创建新的服务账户。
3. 为服务账户命名并分配角色为 Viewer。
4. 点击 Add service account token 并点击 Generate token。
5. 复制生成的令牌。
6. 这将用作提供者配置中的 `service_account_token` 参数。

## 创建和更新 Grafana Incidents

Grafana Incident 提供者支持在 Grafana 中创建和更新事件。

- `operationType` - 操作类型可以是 `create` 或 `update`。
- `updateType` - 更新类型用于更新事件的各个字段。

### 创建事件

- `operationType` - `create`
- `title` (str) - 事件的标题。
- `severity` (str) - 事件的严重程度。
- `labels` (list) - 事件的标签。
- `roomPrefix` (str) - 事件的房间前缀。
- `isDrill` (bool) - 事件的演练状态。
- `status` (str) - 事件的状态。
- `attachCaption` (str) - 事件的附件说明。
- `attachURL` (str) - 事件的附件 URL。

### 更新事件

- `operationType` - `update`
- `updateType` - 可更新的字段有 `removeLabel`、`unassignLabel`、`unassignLabelByUUID`、`unassignRole`、`updateIncidentEventTime`、`updateIncidentIsDrill`、`updateIncidentSeverity`、`updateIncidentStatus`、`updateIncidentTitle`。

#### 移除标签
- `incident_id` (str) - 事件 ID。
- `label` (str) - 要移除的标签。

#### 取消分配标签
- `incident_id` (str) - 事件 ID。
- `label` (str) - 要取消分配的标签。
- `key` (str) - 要取消分配的标签的键。

#### 按 UUID 取消分配标签
- `incident_id` (str) - 事件 ID。
- `key_uuid` (str) - 要取消分配的标签的键 UUID。
- `value_uuid` (str) - 要取消分配的标签的值 UUID。

#### 取消分配角色
- `incident_id` (str) - 事件 ID。
- `role` (str) - 要取消分配的角色。
- `user_id` (str) - 要取消分配的用户 ID。

#### 更新事件时间
- `incident_id` (str) - 事件 ID。
- `event_time` (str) - 要更新的事件时间。
- `event_name` (str) - 要更新的事件名称。

#### 更新事件演练状态
- `incident_id` (str) - 事件 ID。
- `isDrill` (bool) - 要更新的演练状态。

#### 更新事件严重程度
- `incident_id` (str) - 事件 ID。
- `severity` (str) - 要更新的严重程度。

#### 更新事件状态
- `incident_id` (str) - 事件 ID。
- `status` (str) - 要更新的状态。

#### 更新事件标题
- `incident_id` (str) - 事件 ID。
- `title` (str) - 要更新的标题。

## 连接提供者

在 GitHub 中创建你的 personal access token（经典版）：
- 在任何页面的右上角，点击你的个人资料照片，然后点击 **Settings**。
- 在左侧边栏中，点击 **Developer settings**。
- 在左侧边栏的 Personal access tokens 下，点击 **Tokens (classic)**。
- 选择 Generate new token，然后点击 Generate new **token (classic)**。
- 在 "Note" 字段中，为你的令牌起一个描述性名称。
- 要为令牌设置过期时间，选择 **Expiration**，然后选择默认选项或点击 **Custom** 输入日期。
- 选择你想要授予此令牌的范围。
- 点击 **Generate token**。
- 可选地，点击复制按钮将新令牌复制到剪贴板。

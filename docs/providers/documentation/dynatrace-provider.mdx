## 连接提供者

1. 登录你的 Dynatrace 账户，导航到 "Settings" → "Integration" → "Dynatrace API"。
2. 生成具有适当权限的 API 令牌（例如，Read metrics）。
3. 获取你环境的 Dynatrace URL。
4. 使用 API 令牌和 Dynatrace URL 配置 Dynatrace 提供者。

## 集成密钥生成

Flashduty 使用集成密钥作为认证方法

1. 进入 Flashduty 控制台，选择 集成中心 => 告警事件 进入集成选择页面

![Flashduty](/images/flashduty_2.png)

2. 选择 Keep 集成
3. 为当前集成定义名称
4. 配置默认路由并选择相应的通道
5. 将集成密钥复制到 Keep
6. 完成集成配置

![Flashduty](/images/flashduty_3.png)

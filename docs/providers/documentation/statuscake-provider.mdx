## 连接 Provider

获取 StatusCake API Key

1. 在 [StatusCake](https://www.statuscake.com/) 上创建账户。
2. 登录后，前往 [Account Settings](https://app.statuscake.com/User.php) 下的 My Account
3. 在 Manage API Keys 下，生成新的 API 密钥或使用默认密钥。
4. 复制 API Key。这将作为 Provider 设置中的 `Statuscake API Key` 使用。

## 连接提供者

要连接到该提供者，你可以实例化 `HttpProvider` 类，提供 `provider_id` 和 `ProviderConfig` 对象。然后你可以调用 `query` 方法来查询 HTTP 端点。

## 注意事项

代码会记录一些关于正在发送的请求的调试信息，包括请求标头、正文和查询参数。这些信息不应包含敏感信息，但在生产环境中使用此提供者之前，确保这一点很重要。

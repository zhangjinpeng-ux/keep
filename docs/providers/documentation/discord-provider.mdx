## 连接提供者

- 打开你想要创建 webhook 的 Discord 服务器。
- 点击服务器名称旁边的设置图标，选择 "Server Settings"。
- 在左侧菜单中，点击 "Integrations"，然后点击 "Webhooks"。
- 点击 "Create Webhook" 按钮，为你的 webhook 命名。

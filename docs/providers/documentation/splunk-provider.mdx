## 连接 Provider

获取 Splunk API Token：
1. 确保您拥有具有必要[权限](https://docs.splunk.com/Documentation/Splunk/9.2.0/Security/Rolesandcapabilities)的 Splunk 账户。所需的基本权限是 `list_all_objects` 和 `edit_own_objects`。
2. 获取用于认证 API 请求的 API token。[了解更多](https://docs.splunk.com/Documentation/Splunk/9.2.0/Security/Setupauthenticationwithtokens)关于如何设置和获取 API Keys。

确定您的 Splunk 实例详情：
1. 确定您要连接的 Splunk 实例的主机（IP 地址或主机名）和端口（Splunk 管理 API 的默认端口是 8089）。

---
**注意**
确保按照此[指南](https://docs.splunk.com/Documentation/Splunk/9.2.0/Alert/ConfigureWebhookAllowList)配置 webhook 允许列表，以允许您的 `keep` 部署。

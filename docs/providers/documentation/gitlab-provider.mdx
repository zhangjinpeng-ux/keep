## 连接提供者

1. 前往 [Personal Access Token](https://docs.gitlab.com/ee/user/profile/personal_access_tokens.html#create-a-personal-access-token) 了解如何创建 personal_access_token。
2. 获取 `host`，例如：如果你使用 Cloud GitLab，请使用：`https://gitlab.com`，或者如果你使用本地部署版，请使用你的 `host`。

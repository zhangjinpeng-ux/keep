## 连接提供者

Auth0 提供者连接到 **Authentication API** 和 **Management API**，支持基于令牌的认证和用户管理等功能。根据你的需求，你可以：
- 使用 **Authentication API** 获取访问令牌、管理用户配置文件或处理多因素认证。
- 使用 **Management API** 自动化配置你的 Auth0 环境、注册应用程序、管理用户等。

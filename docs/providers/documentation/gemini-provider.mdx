## 连接提供者

要连接到 Gemini，你需要获取一个 API Key：

1. 前往 [Google AI Studio](https://makersuite.google.com/app/apikey)。
2. 点击 **Create API Key** 或使用已有的。
3. 复制你的 API Key 用于 Keep。

在 Gemini Provider 配置的 `authentication` 部分中使用生成的 API Key。

## 连接提供者

要连接到 Axiom，你需要从 Axiom 账户创建 API 令牌。按照以下步骤操作：

1. 登录你的 Axiom 账户。
2. 在 **Settings** 菜单下转到 **API Access** 页面。
3. 点击 **Create Token** 按钮并输入令牌名称。
4. 复制令牌值并妥善保管。
5. 将令牌值添加到 Axiom Provider 配置的 `authentication` 部分。

要访问数据集，你需要提供组织 ID。你可以在 Axiom Web 应用的 URL 中找到组织 ID。例如，如果你的 Axiom URL 是 `https://app.axiom.co/organizations/1234`，那么你的组织 ID 是 `1234`。

## 注意事项

- 此提供者支持 Axiom API 提供的有限功能集。
- `startTime` 和 `endTime` 参数使用 ISO-8601 格式。
- `query` 函数以 JSON 格式从 Axiom API 返回响应。

## 连接 Provider

1. 在 [Rollbar](https://rollbar.com/) 上创建账户。
2. 登录后，导航到您要连接的项目，进入项目设置。
3. 在 Setup 下，前往 Project Access Tokens，创建具有读写权限的新令牌。
4. 复制生成的令牌。
5. 这将作为 Provider 配置中的 `rollbarAccessToken` 参数使用。

## Webhook 集成修改

您可以通过在 Rollbar 项目中导航到 **Settings > Notifications > Webhook** 来管理 webhook 集成授予的权限。

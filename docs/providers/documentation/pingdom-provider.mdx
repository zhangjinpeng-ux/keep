## 连接 Provider

### API Key

要获取 Pingdom API 密钥，请按照以下步骤操作：

1. 登录您的 Pingdom 账户。
2. 导航到 "Settings" 部分。
3. 点击 "Pingdom API" 选项卡。
4. 生成新的 API Key。


## 指纹识别

Pingdom 中的指纹是根据传入/拉取的 `check_id` 事件计算的。

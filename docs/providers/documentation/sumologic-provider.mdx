## 概述

SumoLogic provider 通过 Webhook Connection 实现从 SumoLogic 中的 Monitor 接收告警。

<AutoGeneratedSnippet />

## 连接 Provider

1. 按照[此处的说明](https://help.sumologic.com/docs/manage/security/access-keys/)获取您的 Access Key 和 Access ID
2. 确保用户拥有具有以下功能的角色：
    - `manageScheduledViews`
    - `manageConnections`
    - `manageUsersAndRoles`
3. 从[此处](https://api.sumologic.com/docs/#section/Getting-Started/API-Endpoints)查找您的 `deployment`，Keep 将自动确定您的端点。

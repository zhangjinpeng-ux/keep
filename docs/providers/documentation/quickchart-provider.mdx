# QuickChart Provider

## 概述

QuickChart provider 允许基于 Keep 平台内的告警数据生成两种类型的图表：

1. 折线图，显示特定指纹告警随时间变化的趋势。
2. 径向仪表图，显示 Keep 收到的该指纹告警总数。

这些图表可用于各种报告、仪表板或告警摘要中，以提供告警活动和趋势的可视化洞察。

## 连接 Provider

### 不使用 API 密钥使用 QuickChart

QuickChart provider 可以在不需要 API 密钥的情况下生成图表。但是，这种使用方式仅限于基本功能和较低的请求限制。

### 使用 API 密钥使用 QuickChart

要解锁更高级的功能和更高的使用限制，您可以使用 QuickChart API 密钥。获取方法如下：

1. 访问 [QuickChart](https://quickchart.io/)。
2. 注册一个免费账户开始使用。
3. 导航到账户设置，找到您的 API 密钥。

获得 API 密钥后，将其添加到 Keep 中的 Provider 配置中。

## 注意事项

此 Provider 旨在在 Keep 中提供灵活的图表生成功能，增强您可视化告警数据和趋势的方式。它非常适合希望将告警活动的可视化表示快速集成到工作流中的用户。

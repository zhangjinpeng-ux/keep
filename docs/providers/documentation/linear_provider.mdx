## 如何设置

Linear Provider 使用 `api_token` 进行请求授权。你需要提供以下内容：

- **api_token**（必需）：你的 Linear 应用的个人 API 密钥。
  - 如何获取：
    1. 访问 Linear 应用或网站。
    2. 登录你的 Linear 账户。
    3. 导航到你的账户设置。
    4. 导航到 API 页面。
    5. 在 Personal API keys 部分下生成密钥。
    6. 复制生成的 API 令牌。

## 注意事项

- 此提供者允许你查询给定 Linear 团队的项目。
- 此提供者允许你在 Linear 应用中为给定项目和团队通知（创建 issue）。

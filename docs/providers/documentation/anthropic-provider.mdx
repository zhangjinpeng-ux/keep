<Tip>
  Anthropic Provider 支持查询 Claude 语言模型进行基于提示的交互。
</Tip>

## 输出

目前，Claude Provider 根据提供的提示输出模型的响应。

## 连接提供者

要连接到 Claude，你需要获取一个 API Key：

1. 登录你的 Anthropic 账户：[Anthropic Console](https://console.anthropic.com)。
2. 导航到 **API Keys** 部分。
3. 点击 **Create Key** 为 Keep 生成新的 API 密钥。

在 Claude Provider 配置的 `authentication` 部分中使用生成的 API 密钥。

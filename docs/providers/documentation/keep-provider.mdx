## 认证参数

要使用 Keep 提供者，你必须使用与 Keep 账户关联的 API 令牌进行认证。此令牌可以从你的 Keep 仪表板生成。

## 连接提供者

1. 登录你的 Keep 账户。
2. 导航到账户仪表板的 API 部分并生成 API 令牌。
3. 在通过 Keep 提供者查询告警时使用此令牌进行认证。

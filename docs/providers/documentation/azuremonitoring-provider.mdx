## 概述

Azure Monitor Provider 将 Keep 与 Azure Monitor 集成，使你能够在 Keep 平台内接收告警。通过在 Azure 中设置 webhook，你可以确保关键告警被发送到 Keep，从而实现高效的监控和响应。

## 连接 Azure Monitor 到 Keep

将 Azure Monitor 连接到 Keep 涉及在 Azure 中创建 Action Group、添加 webhook 操作以及配置 Alert Rule 以使用新的 Action Group。

### 步骤 1：导航到 Action Group
1. 登录你的 Azure 门户。
2. 导航到 **Monitor** > **Alerts** > **Action groups**。

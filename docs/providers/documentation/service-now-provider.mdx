## 连接 Provider

1. 确保 ServiceNow 实例可通过 API 访问。
2. 在 Provider 配置中提供必要的 API 凭据（`instance_url` 和 `api_token`）。

## 附加说明

- 后端环境变量 `KEEP_SERVICENOW_PROVIDER_SKIP_SCOPE_VALIDATION` 允许绕过范围验证。

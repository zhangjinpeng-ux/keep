## 连接提供者

要连接到 MySQL 数据库，你需要创建一个具有所需权限的新用户。以下是你可以执行此操作的方法：

1. 以具有足够权限创建新用户的用户身份连接到 MySQL 服务器。
2. 运行以下命令创建新用户：
   `CREATE USER '<username>'@'<host>' IDENTIFIED BY '<password>'`;
3. 通过运行以下命令授予新用户必要的权限：
   `GRANT ALL PRIVILEGES ON <database>.* TO '<username>'@'<host>'`;

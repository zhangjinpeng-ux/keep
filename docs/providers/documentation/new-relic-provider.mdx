## 连接 Provider

1. 访问 https://one.newrelic.com/admin-portal/api-keys/home 创建用户密钥。
2. 从创建的密钥中获取 `api_key` 和 `account_id`。
3. 根据您所在的区域，从此处 https://docs.newrelic.com/docs/apis/rest-api-v2/get-started/introduction-new-relic-rest-api-v2 获取 `api_url`。

## Webhook 集成修改

webhook 集成将 Keep 添加为 New Relic 中"Alerts and AI" API 的目标地址。
这赋予 Keep 在 New Relic 中访问以下权限范围的能力：
- `ai.destinations:read`
- `ai.destinations:write`
- `ai.channels:read`
- `ai.channels:write`

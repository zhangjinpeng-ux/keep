## 连接 Provider

1. 设置一个 Prometheus 服务器，确保其正在运行。
2. 获取您的 Prometheus 实例可访问的 `prometheus_url`。
3. （可选）如果 Prometheus 配置了保护，请获取 API 令牌。
4. 在 Provider 配置中提供这些值。

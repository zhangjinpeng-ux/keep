## 连接 Provider

- 要与 Pagertree API 交互，您需要提供 api_token。
- 您可以在 [User Settings](https://app.pagertree.com/user/settings) 页面查看和管理您的 API 密钥。


## 注意事项

_此 Provider 使用 Pagertree API 根据提供的参数发送告警或将其标记为事件。根据是否将 incident 标记为 true，它分别调用 `__send_alert` 或 `__send_incident` 方法。_

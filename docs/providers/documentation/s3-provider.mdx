## 限制

仅支持查询 yaml、yml、json、xml 和 csv 文件。

## 权限范围

请注意，在安装过程中，Provider 会执行 `list_buckets` 来验证配置。以下是示例 IAM 策略：
```
{
	"Version": "2025-01-15",
	"Statement": [
		{
			"Sid": "VisualEditor0",
			"Effect": "Allow",
			"Action": [
				"s3:ListBucket",
				"s3:GetObject",
				"s3:GetBucketLocation",
				"s3:ListAllMyBuckets"
			],
			"Resource": "*"
		}
	]
}
```

## 连接提供者

### API Key

要获取 Datadog API Key，请按照以下步骤操作：

1. 登录你的 Datadog 账户。
2. 导航到 "Integrations" 部分。
3. 点击 "API" 标签页。
4. 生成新的 API Key。

### App Key

要获取 Datadog App Key，请按照以下步骤操作：

1. 登录你的 Datadog 账户。
2. 导航到 "Integrations" 部分。
3. 点击 "API" 标签页。
4. 生成新的 App Key 或使用已有的。

## 指纹

Datadog 中的指纹是基于传入/拉取事件的 `groups` 和 `monitor_id` 字段计算的。

## Webhook 集成修改

Webhook 集成将 Keep 作为监视器添加到 Datadog 中。可以在 "Monitors" 部分下找到它。
该集成自动获得 Datadog 中的以下权限范围：
- `monitors_read`
- `monitors_write`
- `create_webhooks`

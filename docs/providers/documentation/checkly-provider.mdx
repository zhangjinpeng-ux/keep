## 连接 Checkly 到 Keep

1. 打开 Checkly 仪表板，点击右上角的个人资料图片。

2. 点击 `User Settings`。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_1.png" />
</Frame>

3. 打开 `API Keys` 标签页，点击 `Create API Key` 生成新的 API 密钥。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_2.png" />
</Frame>

4. 复制 API 密钥。

5. 打开 Account Settings 下的 `General` 标签页，复制 `Account ID`。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_3.png" />
</Frame>

6. 转到 Keep，将 Checkly 添加为提供者，在相应字段中输入 API 密钥和 Account ID，然后点击 `Connect`。

## Webhooks 集成

1. 打开 Checkly 仪表板，打开左侧栏中的 `Alerts` 标签页。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_4.png" />
</Frame>

2. 点击 `Add more channels`

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_5.png" />
</Frame>

3. 从可用通道列表中选择 `Webhook`。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_6.png" />
</Frame>

4. 输入 webhook 名称，选择方法为 `POST`

5. 输入 [https://api.keephq.dev/alerts/event/checkly](https://api.keephq.dev/alerts/event/checkly) 作为 URL。

6. 复制以下代码片段并粘贴到 Webhook 的 `Body` 中。请参考下面的截图。

```json
{
  "event": "{{ALERT_TITLE}}",
  "alert_type": "{{ALERT_TYPE}}",
  "check_name": "{{CHECK_NAME}}",
  "group_name": "{{GROUP_NAME}}",
  "check_id": "{{CHECK_ID}}",
  "check_type": "{{CHECK_TYPE}}",
  "check_result_id": "{{CHECK_RESULT_ID}}",
  "check_error_message": "{{CHECK_ERROR_MESSAGE}}",
  "response_time": "{{RESPONSE_TIME}}",
  "api_check_response_status_code": "{{API_CHECK_RESPONSE_STATUS_CODE}}",
  "api_check_response_status_text": "{{API_CHECK_RESPONSE_STATUS_TEXT}}",
  "run_location": "{{RUN_LOCATION}}",
  "ssl_days_remaining": "{{SSL_DAYS_REMAINING}}",
  "ssl_check_domain": "{{SSL_CHECK_DOMAIN}}",
  "started_at": "{{STARTED_AT}}",
  "tags": "{{TAGS}}",
  "link": "{{RESULT_LINK}}",
  "region": "{{REGION}}",
  "uuid": "{{$UUID}}"
}
```

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_7.png" />
</Frame>

7. 转到 Headers 标签页，添加一个键为 `X-API-KEY` 的新标头，在 Keep 中创建新的 API 密钥并将其粘贴为值，然后保存 webhook。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_8.png" />
</Frame>

8. 按照以下步骤在 Keep 中创建新的 API 密钥。

9. 转到 Keep 仪表板，点击左下角的个人资料图标，然后点击 `Settings`。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_9.png" />
</Frame>

10. 选择 `Users and Access` 标签页，然后选择 `API Keys` 标签页并创建新的 API 密钥。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_10.png" />
</Frame>

11. 命名并选择角色为 `webhook`，然后点击 `Create API Key`。

<Frame width="100" height="200">
  <img height="10" src="/images/checkly-provider_11.png" />
</Frame>

12. 在 Checkly 中创建的 webhook 的 `X-API-KEY` 标头中使用生成的 API 密钥。

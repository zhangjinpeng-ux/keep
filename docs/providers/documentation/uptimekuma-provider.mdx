## 连接 Provider

获取 UptimeKuma Host URL、Username 和 Password

1. UptimeKuma 只能自托管。您需要有一个正在运行的 UptimeKuma 实例。
2. 设置 UptimeKuma 后，您可以获取 Host URL、Username 和 Password。
3. 在 Provider 设置中使用获取的 Host URL、Username 和 Password。

## Webhooks 集成

1. 使用所需参数连接到 UptimeKuma provider。
2. 在 UptimeKuma 中使用 Keep Backend API URL 作为 Host URL。[https://api.keephq.dev](https://api.keephq.dev)（默认）
3. 在 Keep 中导航到 Account Settings，进入 API Keys，为 Webhook 生成 API Key。

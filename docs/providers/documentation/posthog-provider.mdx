## 连接 Provider

### API Key

要获取 PostHog API 密钥，请按照以下步骤操作：

1. 登录您的 PostHog 账户。
2. 导航到 "Project Settings" > "API Keys"。
3. 创建一个新的 API 密钥或使用现有的密钥。
4. 复制 API 密钥值。

### Project ID

要查找您的 PostHog 项目 ID：

1. 登录您的 PostHog 账户。
2. 项目 ID 显示在项目设置中，或者在查看项目时的 URL 中可见。

## 可用方法

PostHog provider 提供以下方法：

### 获取会话录制域名

检索指定时间段内会话录制中的域名列表。

```yaml
- name: get-posthog-domains
  provider:
    config: "{{ providers.posthog }}"
    type: posthog
    with:
      query_type: session_recording_domains
      hours: 24       # Number of hours to look back
      limit: 500      # Maximum number of recordings to fetch
```

### 获取会话录制

检索指定时间段内的会话录制数据。

```yaml
- name: get-posthog-recordings
  provider:
    config: "{{ providers.posthog }}"
    type: posthog
    with:
      query_type: session_recordings
      hours: 24       # Number of hours to look back
      limit: 100      # Maximum number of recordings to fetch
```

## 示例工作流

以下是一个示例工作流，跟踪过去 24 小时内 PostHog 会话录制的域名，并将摘要发送到 Slack：

```yaml
workflow:
  id: posthog-domain-tracker
  name: PostHog Domain Tracker
  description: Tracks domains from PostHog session recordings over the last 24 hours and sends a summary to Slack.
  triggers:
    - type: manual
    - type: interval
      value: 86400  # Run daily (in seconds)
  steps:
    - name: get-posthog-domains
      provider:
        config: "{{ providers.posthog }}"
        type: posthog
        with:
          query_type: session_recording_domains
          hours: 24
          limit: 500
  actions:
      - name: send-to-slack
        provider:
          config: "{{ providers.slack }}"
          type: slack
          with:
            blocks:
              - type: header
                text:
                  type: plain_text
                  text: "PostHog Session Recording Domains (Last 24 Hours)"
                  emoji: true
              - type: section
                text:
                  type: mrkdwn
                  text: "Found *{{ steps.get-posthog-domains.results.unique_domains_count }}* unique domains across *{{ steps.get-posthog-domains.results.total_domains_found }}* occurrences"
              - type: divider
              - type: section
                text:
                  type: mrkdwn
                  text: "Domains:*"
              - type: section
                text:
                  type: mrkdwn
                  text: "{{#steps.get-posthog-domains.results.unique_domains}}
                    • *{{ . }}*
                    {{/steps.get-posthog-domains.results.unique_domains}}"
              - type: divider
```

## 注意事项

PostHog provider 需要以下权限范围：
- `session_recording:read` - 允许读取会话录制数据
- `project:read` - 允许读取项目数据
- `session_recording_playlist:read` - 可选的录制播放列表访问权限

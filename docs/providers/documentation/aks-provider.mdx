## 连接提供者

要连接到 Azure AKS，请按照以下步骤操作：

1. 登录你的 [Azure](https://azure.microsoft.com/) 账户。
2. 转到你的 kubernetes 服务页面，点击 `Connect` 按钮，然后点击 `Open Cloud Shell`。
3. 在 Cloud Shell 中运行 `az ad sp create-for-rbac --role owner --scopes /subscriptions/<YOUR_SUBSCRIPTION_ID>`，你将获得类似以下的响应：
   ```
    {
      "appId": "xxxxxx-xxxxx-xxxxxx-xxxx",
      "displayName": "azure-cli-2023-11-06-13-00-52",
      "password": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
      "tenant": "xxxxx-xxxxx-xxxx-xxxxx"
    }
   ```
   在上面的 JSON 对象中，`appId` 是 `client_id`，`password` 是 `client_secret`，`tenant` 是 `tenant_id`

## 注意事项

- 此提供者允许你与 Azure AKS 交互以查询 kubernetes 集群中的资源。

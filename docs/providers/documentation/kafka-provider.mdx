## 连接提供者

1. 设置一个 Kafka broker（或使用现有的）并确保它可访问。
2. 获取 broker URL（例如，`localhost:9092` 或远程 Kafka 服务 URL）。
3. （可选）如果使用安全通信，请提供安全协议、SASL 机制、用户名和密码。
4. 使用这些参数配置提供者。

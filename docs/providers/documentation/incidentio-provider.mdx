## 概述

Incident.io 提供者支持与 Incident.io API 的交互，允许管理事件。这包括查询特定事件、检索所有事件以及管理事件详细信息。此提供者将 Keep 系统与 Incident.io 强大的事件管理平台集成在一起。

## 连接提供者

### API Key

要使用 Incident.io API：
1. 登录你的 Incident.io 账户。
2. 导航到账户设置下的 "API Keys" 部分。
3. 生成新的 API Key 或使用已有的。
4. 确保它具有启用的 `read` 权限，用于读取和管理事件。

### 事件端点

Incident.io 事件端点允许查询和管理事件。操作包括检索特定事件详细信息或获取所有事件列表。这对于高效监控和响应事件至关重要。

有关更多详细信息，请参考 [Incident.io API 文档](https://api-docs.incident.io/)。

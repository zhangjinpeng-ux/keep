## 连接 Provider

1. 确保您有一个正在运行的 VMAlert 实例，可通过指定的主机和端口访问。
2. 在初始化 Provider 时，将主机和端口信息包含在 Victoriametrics provider 配置中。

## 查询 Victoriametrics

Victoriametrics provider 允许您通过 `query` 和 `query_range` 类型从 Victoriametrics 查询。以下是可用于查询的参数：

1. `query` 类型：

   - `query`：在 Victoriametrics 上执行的查询。示例：`sum(rate(http_requests_total{job="api-server"}[5m]))`。
   - `start`：查询数据的时间。示例：`2024-01-01T00:00:00Z`

2. `query_range` 类型：
   - `query`：在 Victoriametrics 上执行的查询。示例：`sum(rate(http_requests_total{job="api-server"}[5m]))`。
   - `start`：查询数据的开始时间。示例：`2024-01-01T00:00:00Z`
   - `end`：查询数据的结束时间。示例：`2024-01-01T00:00:00Z`
   - `step`：查询的步长。示例：`15s`

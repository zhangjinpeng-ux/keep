## 概述

**ArgoCD 提供者** 支持从 ArgoCD 拉取拓扑和应用程序数据。
ArgoCD Applications 映射到 Keep Services
ArgoCD ApplicationSets 映射到 Keep Applications

<AutoGeneratedSnippet />

## 连接提供者

1. 通过遵循 [ArgoCD 用户管理文档](https://argo-cd.readthedocs.io/en/stable/operator-manual/user-management/#manage-users) 中的 `Generate auth token` 步骤，从你的 ArgoCD 实例获取 **access token**。
2. 将 **deployment URL** 设置为你的 ArgoCD 实例的基础 URL（例如，`https://localhost:8080`）。

## 功能

**ArgoCD 提供者** 支持以下关键功能：

- **拓扑**：使用 ArgoCD 中的 applications 配置拓扑。
- **应用程序**：使用 ArgoCD 中的 ApplicationSets 创建 Applications。

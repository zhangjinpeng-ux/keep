## 连接 Provider

要使用 Signl4Provider，您需要提供您的 signl4_integration_secret。

您可以在 SIGNL4 Web 门户的 **Teams** 或 **Integrations** -> **Distribution Rules** 下找到您的集成或团队密钥。

signl4_integration_secret 用于通过 webhook API 向 SIGNL4 发布事件。

## 注意事项

该 Provider 使用 events API 或 incidents API 来创建告警或事件。选择使用哪个 API 取决于是否存在 routing_key 或 api_key。

## 连接提供者

### 获取 API Token

要使用 LinearB 提供者，你必须从 LinearB 获取一个 API Token：

1. 登录你的 LinearB 账户。
2. 导航到 API 设置部分。
3. 生成具有适当权限的新 API Token。
4. 安全存储 API Token，因为在 Keep 中配置 LinearB 提供者时需要它。

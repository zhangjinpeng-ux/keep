## 连接提供者

### API Key

要获取 Elastic API Key，请按照以下步骤操作：

1. 登录你的 elastic.co 账户
2. 转到 "Elasticsearch Service" 部分
3. 点击 "API Key" 按钮
4. 生成新的 API Key

### Cloud ID

要获取 Elastic Cloud ID，请按照以下步骤操作：

1. 登录你的 elastic.co 账户
2. 转到 "Elasticsearch Service" 部分
3. 在 Overview 页面中找到 "Cloud ID"。

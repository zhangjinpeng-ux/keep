## 连接 Provider

要连接到 Postgres 数据库，您需要创建一个具有所需权限的新用户。操作步骤如下：

1. 以具有足够权限创建新用户的身份连接到 PostgreSQL 服务器。
2. 运行以下命令创建新用户：
   `CREATE USER '<username>' WITH ENCRYPTED PASSWORD '<password>'`;
3. 运行以下命令创建数据库：
   `CREATE DATABASE '<yourdbname>';`;
4. 运行以下命令为新用户授予必要的权限：
   `GRANT ALL PRIVILEGES ON <database>.* TO '<username>'`;

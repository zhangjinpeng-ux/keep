## 将 ThousandEyes 连接到 Keep

1. 前往 [ThousandEyes Dashboard](https://app.thousandeyes.com/dashboard)

2. 点击左侧边栏中的 `Manage`，选择 `Account Settings`。

<Frame width="100" height="200">
  <img height="10" src="/images/thousandeyes-provider_1.png" />
</Frame>

3. 在 Account Settings 中选择 `Users and Roles`

<Frame width="100" height="200">
  <img height="10" src="/images/thousandeyes-provider_2.png" />
</Frame>

4. 在 `User API Tokens` 下，您可以创建 OAuth Bearer Token

<Frame width="100" height="200">
  <img height="10" src="/images/thousandeyes-provider_3.png" />
</Frame>

5. 复制生成的令牌。这将作为 Provider 设置中的 `OAuth2 Bearer Token` 使用。

## Webhooks 集成

1. 打开 [ThousandEyes Dashboard](https://app.thousandeyes.com/dashboard)，点击左侧边栏中的 `Network & App Synthetics`，选择 `Agent Settings`。

<Frame width="100" height="200">
  <img height="10" src="/images/thousandeyes-provider_4.png" />
</Frame>

2. 前往 `Enterprise Agents` 下的 `Notifications`，点击 `Notifications`。

<Frame width="100" height="200">
  <img height="10" src="/images/thousandeyes-provider_5.png" />
</Frame>

3. 前往 `Notifications`，创建新的 webhook 通知。

<Frame width="100" height="200">
  <img height="10" src="/images/thousandeyes-provider_6.png" />
</Frame>

4. 为其命名，设置 URL 为 [https://api.keephq.dev/alerts/event/thousandeyes?api_key=your-api-key](https://api.keephq.dev/alerts/event/thousandeyes?api_key=your-api-key)

5. 选择 `Auth Type` 为 None，然后 `Add New Webhook`。

<Frame width="100" height="200">
  <img height="10" src="/images/thousandeyes-provider_7.png" />
</Frame>

6. 前往 Keep 仪表板，点击左下角的个人资料图标，然后点击 `Settings`。

<Frame width="100" height="200">
  <img height="10" src="/images/keep-apikey-1.png" />
</Frame>

7. 选择 `Users and Access` 选项卡，然后选择 `API Keys` 选项卡，创建一个新的 API 密钥。

<Frame width="100" height="200">
  <img height="10" src="/images/keep-apikey-2.png" />
</Frame>

8. 输入名称，选择角色为 `webhook`，然后点击 `Create API Key`。

<Frame width="100" height="200">
  <img height="10" src="/images/keep-apikey-3.png" />
</Frame>

9. 复制 API 密钥，粘贴到 webhook URL 中。

<Frame width="100" height="200">
  <img height="10" src="/images/keep-apikey-4.png" />
</Frame>

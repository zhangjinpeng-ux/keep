## 连接 Provider

要连接到 Microsoft Planner，请按照以下步骤操作：

1. 登录您的 [Azure](https://azure.microsoft.com/) 账户。
2. 在[此处](https://portal.azure.com/#view/Microsoft_AAD_RegisteredApps/CreateApplicationBlade/isMSAApp~/false)注册一个应用程序。
3. 成功注册应用程序后，前往 **API permissions** 页面，添加以下权限：
    - `Tasks.Read.All`
    - `Tasks.ReadWrite.All`
4. 前往 **Overview** 页面，记下 `Application (client) ID` 和 `Directory (tenant) ID`。
5. 前往 **Certificates & secrets** 页面，创建一个新的客户端密钥，记下客户端密钥值。
6. 将 client id、client secret 和 tenant id 添加到 Microsoft Planner Provider 配置的 `authentication` 部分。

## 注意事项

- 此 Provider 允许您与 Microsoft Planner Provider 交互以创建任务。

  Llama.cpp Provider 支持查询本地 Llama.cpp 模型进行基于提示的交互。请确保你已在本地运行 Llama.cpp 服务器并加载了你想要的模型。

### **云端限制**
此提供者在云环境中被禁用，只能在本地或自托管环境中使用。

## 连接提供者

要使用 Llama.cpp Provider：

1. 在你的系统上安装 Llama.cpp
2. 下载或将你的模型转换为 GGUF 格式
3. 使用 HTTP 接口启动 Llama.cpp 服务器：
   ```bash
   ./server --model /path/to/your/model.gguf --host 0.0.0.0 --port 8080
   ```
4. 在你的 Keep 配置中配置主机 URL 和模型路径

## 前提条件

- Llama.cpp 必须安装并编译了服务器支持
- 系统上必须有一个 GGUF 格式的模型文件
- Llama.cpp 服务器必须正在运行且可访问
- 服务器必须有足够的资源来加载和运行你的模型

## 模型兼容性

提供者适用于任何与 Llama.cpp 兼容的 GGUF 格式模型，包括：
- LLaMA 和 LLaMA-2 模型
- Mistral 模型
- OpenLLaMA 模型
- Vicuna 模型
- 以及其他兼容的模型架构

在使用提供者之前，请确保你的模型是 GGUF 格式。

## 连接提供者

要连接到 DeepSeek，你需要获取一个 API Key：

1. 在 [DeepSeek](https://platform.deepseek.com) 注册账户
2. 导航到你的账户设置
3. 为 Keep 生成一个 API Key

在 DeepSeek Provider 配置的 `authentication` 部分中使用生成的 API Key。

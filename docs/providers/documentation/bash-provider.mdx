## 连接提供者

Bash 提供者允许你在工作流中运行 Bash 命令或脚本。你可以传入任何有效的 Bash 命令，它将在本地环境中执行。

### **云端限制**
此提供者在云环境中被禁用，只能在本地或自托管环境中使用。

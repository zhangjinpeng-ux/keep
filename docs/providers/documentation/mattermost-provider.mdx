## 连接提供者

1. **获取 Mattermost Webhook URL：**
   - 前往 Mattermost Incoming Webhook API 文档：[Mattermost Incoming Webhooks](https://docs.mattermost.com/developer/webhooks-incoming.html)。
   - 按照说明创建新的 incoming webhook。
   - 复制生成的 webhook URL，应将其作为 `webhook_url` 用于认证。

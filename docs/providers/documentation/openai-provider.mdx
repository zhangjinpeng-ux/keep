## 连接 Provider

要连接到 OpenAI，您需要获取 API 密钥和（可选）组织 ID：

1. 在 [OpenAI Platform](https://platform.openai.com) 登录您的 OpenAI 账户。
2. 前往 **API Keys** 部分。
3. 点击 **Create new secret key** 为 Keep 生成密钥。
4. （可选）如果您属于多个组织，请在 **Organization settings** 下获取您的 **Organization ID**。

在 OpenAI Provider 配置的 `authentication` 部分中使用生成的 API 密钥。

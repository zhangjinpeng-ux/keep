此工作流演示了如何仅在指定的工作时间内执行操作。


说明：
- 触发器：由告警触发或手动触发。
- 动作：检查当前时间是否在 `America/New_York` 时区的工作时间范围内。如果是，则增强告警，将 `businesshours` 字段设置为 `true`。


```yaml
workflow:
  id: businesshours
  description: demonstrate how to do smth only when it's business hours
  triggers:
    - type: alert
    - type: manual
  actions:
    - name: dismiss-alert
      if: "keep.is_business_hours(timezone='America/New_York')"
      provider:
        type: mock
        with:
          enrich_alert:
            - key: businesshours
              value: "true"

```

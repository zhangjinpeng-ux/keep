title: "为告警创建 ServiceNow 工单"
---

<Info>

链接到[工作流](https://github.com/keephq/keep/blob/main/examples/workflows/create_service_now_ticket_upon_alerts.yml)。

</Info>

当来自 Grafana 或 Prometheus 的告警触发时，此工作流会创建一个 ServiceNow 工单。


说明：
- 触发器：由来自 Grafana 或 Prometheus 的告警触发。
- 动作：如果告警还没有工单 ID，则创建 ServiceNow 工单，并使用工单 ID、URL 和状态等详细信息增强告警。


```yaml
workflow:
  id: servicenow
  description: create a ticket in servicenow when an alert is triggered
  triggers:
    - type: alert
      cel: source.contains("grafana") || source.contains("prometheus")
  actions:
    - name: create-service-now-ticket
      if: "not '{{ alert.ticket_id }}' and {{ alert.annotations.ticket_type }}"
      provider:
        type: servicenow
        config: "{{ providers.servicenow }}"
        with:
          table_name: "{{ alert.annotations.ticket_type }}"
          payload:
            short_description: "{{ alert.name }} - {{ alert.description }} [created by Keep][fingerprint: {{alert.fingerprint}}]"
            description: "{{ alert.description }}"
          enrich_alert:
            - key: ticket_type
              value: servicenow
            - key: ticket_id
              value: results.sys_id
            - key: ticket_url
              value: results.link
            - key: ticket_status
              value: results.stage

```

---
title: "函数"
---

Keep 工作流引擎中的**函数（Functions）**是可用于在工作流中操作数据、检查条件或执行转换的实用工具。本文档提供了每个可用函数的简要概述和使用示例。

---

## 数学函数

### `add`

**描述：** 将所有提供的数字相加。所有参数会被转换为整数。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.add(1, 2, 3) # Output: 6
        message2: keep.add(10, 20, 30) # Output: 60
```

---

### `sub`

**描述：** 从第一个数字中减去所有后续数字。所有参数会被转换为整数。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.sub(10, 2, 3) # Output: 5
        message2: keep.sub(100, 20, 30) # Output: 50
```

---

### `mul`

**描述：** 将所有提供的数字相乘。所有参数会被转换为整数。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.mul(2, 3, 4) # Output: 24
        message2: keep.mul(5, 6, 7) # Output: 210
```

---

### `div`

**描述：** 将第一个数字依次除以所有后续数字。所有参数会被转换为整数。如果除法结果是整数则返回整数，否则返回浮点数。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.div(10, 2) # Output: 5
        message2: keep.div(10, 3) # Output: 3.3333333333333335
        message3: keep.div(100, 2, 5) # Output: 10
```

---

### `mod`

**描述：** 计算第一个数字依次除以所有后续数字的余数。所有参数会被转换为整数。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.mod(10, 3) # Output: 1
        message2: keep.mod(100, 30, 7) # Output: 2
```

---

### `exp`

**描述：** 将第一个数字的幂次方提升到所有后续数字乘积的值。所有参数会被转换为整数。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.exp(2, 3) # Output: 8
        message2: keep.exp(2, 3, 2) # Output: 64
```

---

### `fdiv`

**描述：** 将第一个数字依次除以所有后续数字并执行整数除法。所有参数会被转换为整数。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.fdiv(10, 3) # Output: 3
        message2: keep.fdiv(100, 3, 2) # Output: 16
```

---

### `eq`

**描述：** 检查两个值是否相等。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.eq(5, 5) # Output: true
        message2: keep.eq("hello", "world") # Output: false
        message3: keep.eq([1, 2, 3], [1, 2, 3]) # Output: true
```

## 字符串函数

### `uppercase`

**描述：** 将字符串转换为大写。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: "keep.uppercase('hello world')" # Output: "HELLO WORLD"
```

---

### `lowercase`

**描述：** 将字符串转换为小写。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: "keep.lowercase('HELLO WORLD')" # Output: "hello world"
```

---

### `capitalize`

**描述：** 将字符串的首字符大写。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.capitalize("hello world") # Output: "Hello world"
```

---

### `title`

**描述：** 将字符串转换为标题大小写（每个单词首字母大写）。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.title("hello world") # Output: "Hello World"
```

---

### `split`

**描述：** 使用分隔符将字符串拆分为列表。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: "keep.split('a,b,c', ',')" # Output: ["a", "b", "c"]
```

---

### `strip`

**描述：** 移除字符串开头和结尾的空白字符。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.strip("  hello world  ") # Output: "hello world"
```

---

### `replace`

**描述：** 将字符串中的子串替换为另一个字符串。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.replace("hello world", "world", "Keep") # Output: "hello Keep"
```

---

### `remove_newlines`

**描述：** 移除字符串中的所有换行符和制表符。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.remove_newlines("hello\nworld\t!") # Output: "helloworld!"
```

---

### `encode`

**描述：** 对字符串进行 URL 编码。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.encode("hello world") # Output: "hello%20world"
```

---

### `slice`

**描述：** 基于起始和结束索引提取字符串的一部分。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.slice("hello world", 0, 5) # Output: "hello"
```

---

### `dict_pop`

**描述：** 从字典中移除指定的键。
**示例：**

```yaml

  - name: example-step
    provider:
      type: mock
      with:
        message: keep.dict_pop({"a": 1, "b": 2, "c": 3}, "a", "b") # Output: {"c": 3}
```

---

### `dict_pop_prefix`

**描述：** 从字典中移除所有以指定前缀开头的键。
**示例：**

```yaml

  - name: example-step
    provider:
      type: mock
      with:
        message: keep.dict_pop_prefix({"a_1": 1, "a_2": 2, "b_1": 3}, "a_") # Output: {"b_1": 3}
```

---

### `dict_filter_by_prefix`

**描述：** 仅返回键以指定前缀开头的字典条目。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.dict_filter_by_prefix({"a_1": 1, "a_2": 2, "b_1": 3}, "a_") # Output: {"a_1": 1, "a_2": 2}
```

---

### `dictget`

**描述：** 从字典中获取值，并带有默认回退值。
**示例：**

```yaml

  - name: example-step

      type: mock
      with:
        message: keep.dictget({"a": 1, "b": 2}, "c", "default") # Output: "default"
```

## 日期和时间函数

### `from_timestamp`

**描述：** 将 Unix 时间戳（整数、浮点数或字符串）转换为 datetime 对象，可附带可选的时区选项。

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: console
      with:
        message: keep.from_timestamp(1717244449.0) # 将打印 "2024-06-01 12:20:49+00:00"
        # 或带时区
        # message: keep.from_timestamp(1717244449.0, "Europe/Berlin") # 将打印 "2024-06-01 14:20:49+02:00"
```

### `utcnow`

**描述：** 返回当前的 UTC 日期时间。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.utcnow()
```

---

### `utcnowtimestamp`

**描述：** 将当前 UTC 日期时间以 Unix 时间戳（自纪元以来的秒数）返回。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.utcnowtimestamp() # Output: 1704067200
```

---

### `utcnowiso`

**描述：** 以 ISO 格式返回当前 UTC 日期时间。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.utcnowiso()
```

---

### `to_utc`

**描述：** 将日期时间字符串或对象转换为 UTC。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.to_utc("2024-01-01T00:00:00")
```

---

### `to_timestamp`

**描述：** 将 datetime 对象或字符串转换为 Unix 时间戳。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.to_timestamp("2024-01-01T00:00:00")
```

---

### `datetime_compare`

**描述：** 比较两个 datetime 对象并以小时为单位返回差值。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.datetime_compare("2024-01-01T10:00:00", "2024-01-01T00:00:00") # Output: 10.0
```

---

### `is_business_hours`

**描述：** 检查给定时间是否在工作时间范围内。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.is_business_hours(
          time_to_check="2024-01-01T14:00:00Z",
          start_hour=8,
          end_hour=20,
          business_days=[0,1,2,3,4],
          timezone="America/New_York"
        )
```

## 实用函数

### `get_firing_time`

**描述：** 以指定时间单位计算告警的触发持续时间。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.get_firing_time(alert, "m", tenant_id="tenant-id") # Output: "15.0"
```

---

### `add_time_to_date`

**描述：** 基于指定时间单位为日期字符串添加时间。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.add_time_to_date("2024-01-01", "%Y-%m-%d", "1w 2d") # Output: "2024-01-10"
```

---

### `timestamp_delta`

**描述：** 向日期时间添加或减去一个时间增量。使用负值来减去时间。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        # 在当前时间上加 2 小时
        add_hours: keep.timestamp_delta(keep.utcnow(), 2, "hours")

        # 从指定日期时间减去 30 分钟
        subtract_minutes: keep.timestamp_delta("2024-01-01T12:00:00Z", -30, "minutes") # Output: 2024-01-01T11:30:00Z

        # 在日期时间上加 1 周
        add_week: keep.timestamp_delta("2024-01-01T00:00:00Z", 1, "weeks") # Output: 2024-01-08T00:00:00Z
```

---

### `is_first_time`

**描述：** 检查具有给定指纹的告警是否是首次触发，或是否在指定时间段内首次触发。
**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        # 检查该告警是否是首次触发
        first_time: keep.is_first_time(alert.fingerprint, tenant_id="tenant-id")

        # 检查该告警是否在过去 24 小时内首次触发
        first_time_24h: keep.is_first_time(alert.fingerprint, "24h", tenant_id="tenant-id")
```

---

**示例：**

```yaml

  - name: example-step

```

---

**示例：**

```yaml
steps:
  - name: example-step
    provider:
      type: mock
      with:
        message: keep.diff([1, 2, 1]) # Output: true
```
